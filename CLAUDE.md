## Project Overview

**AmeyemGeo** is the company portal and geoscience consulting platform for Ameyem. Domain: **ameyem.com** (currently on Replit, needs VPS migration). This is the central hub that links to all Ameyem products.

**Status**: Production-ready, 29/29 Playwright tests passing. Needs migration from Replit to VPS.

## Architecture

```
AmeyemGeo/
├── client/src/             # React 18 frontend (Vite)
│   ├── App.tsx
│   ├── main.tsx
│   ├── components/         # Radix UI + shadcn/ui components
│   ├── pages/              # Page components
│   ├── hooks/              # Custom React hooks
│   └── lib/                # Utilities (queryClient, utils)
├── server/                 # Express.js backend
│   ├── index.ts            # Entry point, Express setup
│   ├── routes.ts           # API routes
│   ├── storage.ts          # Session/storage logic
│   └── vite.ts             # Vite SSR middleware (dev)
├── shared/                 # Shared types & Drizzle schema
│   └── schema.ts           # Database schema + Zod validators
├── tests/e2e/              # Playwright E2E tests (29 tests)
│   ├── 01-pages.spec.cjs        # Page loading (5 tests)
│   ├── 02-navigation.spec.cjs   # Navigation (3 tests)
│   └── 03-visual.spec.cjs       # Visual/UI/Perf (5+ tests)
├── migrations/             # Drizzle auto-generated migrations
└── dist/                   # Production output
```

## Commands

```bash
pnpm dev              # Start dev server (Vite + Express, port 5173 frontend / 5000 backend)
pnpm build            # Production build: Vite → dist/public/, esbuild → dist/index.js
pnpm start            # Run production server (NODE_ENV=production, port 5000)
pnpm check            # TypeScript type checking
pnpm db:push          # Push Drizzle schema to PostgreSQL
npx playwright test   # Run all 29 E2E tests
npx playwright test --ui  # Interactive test runner
```

## Package Manager

**pnpm** only. Never use npm/yarn.

## Command Execution

**PowerShell is mandatory.** All commands must be executed via `powershell.exe -Command "..."` format. Example:
```bash
powershell.exe -Command "cd 'C:\\ArunApps\\AmeyemGeo'; pnpm build"
```

## Quality Gates (MANDATORY before deployment)

```bash
pnpm build && pnpm check && npx playwright test
```

**ALL gates must pass before deployment. No exceptions.**

### Deployment Procedure

1. All quality gates pass (see above)
2. Agent reports `READY TO DEPLOY` with gate results to WTM
3. WTM validates and routes to ServerMigrationAgent
4. SMA deploys via SSH to VPS:
   ```
   cd /opt/AmeyemGeo
   git pull origin main
   pnpm install --frozen-lockfile
   pnpm build
   pnpm db:push
   pm2 start dist/index.js --name ameyem-portal
   # Configure Nginx reverse proxy for ameyem.com → localhost:5000
   ```
5. SMA runs health check: `curl https://ameyem.com`
6. **Never deploy directly from this project pane**

### Rollback Plan
```bash
cd /opt/AmeyemGeo
git log --oneline -5
git revert HEAD
pnpm install --frozen-lockfile
pnpm build
pm2 restart ameyem-portal
```
